# ראיון אונבורדינג — שאלות ואיסוף פרופיל

## מטרה
בנית פרופיל מלא שמאפשר לראוטר לשלוף את מודולי הידע הנכונים.

## שאלות חובה (כולן נדרשות)

### 1. נתוני בסיס
- גיל
- מגדר
- גובה (ס"מ)
- משקל (ק"ג)
- רמת פעילות: יושבני / קלה / בינונית / גבוהה / ספורטאי

### 2. מטרה עיקרית
- ירידה במשקל
- עלייה במסת שריר
- שיפור ביצועים
- אכילה בריאה / אנרגיה
- מצב רפואי ספציפי

### 3. מצבים רפואיים
*שאלה: "האם יש לך מצב רפואי שמשפיע על אכילתך?"*
- סוכרת (סוג 1 / 2)
- מחלות לב / דיסליפידמיה
- הפרעות עיכול (IBS, צליאק, GERD, קרון, קוליטיס)
- בעיות כליות / כבד
- בלוטת תריס (תת/יתר)
- אחר

### 4. אלרגיות ואי-סבילות
- חלב / לקטוז
- גלוטן / חיטה
- ביצים
- אגוזים / בוטנים
- סויה
- אחר

### 5. העדפות וסגנון אכילה
- כשרות (דרגת שמירה)
- צמחוני / טבעוני
- העדפות: ים-תיכוני / אסייתי / ישראלי
- מזונות שלא אוהב

### 6. ספורט ופעילות
- סוג ספורט / פעילות
- תדירות (ימים/שבוע)
- משך אימון (דקות)
- עצימות: קלה / בינונית / עצימה
- מטרת ספורט: בריאות / תחרות / משקל

### 7. שגרת אכילה
- כמה ארוחות ביום?
- האם מדלג על ארוחות?
- האם מנסה IF?
- מי מבשל — עצמאי / בחוץ / שניהם?
